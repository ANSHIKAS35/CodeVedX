

**Project Name:** Utility Usage Prediction Tool

**Project Overview:**

The Utility Usage Prediction Tool is a Python-based console application that helps users manage and predict utility consumption data. The application allows users to add, view, update, and predict monthly utility usage using Machine Learning. Data is stored in a CSV file, making it easy to save and retrieve records. The prediction feature uses the **Linear Regression** algorithm from the **Scikit-learn** library to estimate the next month's utility usage based on previous data.

**Technologies Used:**

* Python
* Pandas
* Scikit-learn
* CSV File Handling

**Project Features:**

* Add monthly utility usage data
* View all stored records
* Update existing records
* Predict next month's utility usage using Linear Regression
* Exception handling for invalid inputs and missing files
* Menu-driven console interface

**Machine Learning Concept Used:**

* Supervised Learning
* Linear Regression

**Learning Outcomes:**

* Python programming fundamentals
* File handling using CSV
* Data processing with Pandas
* Machine Learning using Scikit-learn
* Exception handling
* Menu-driven application development
